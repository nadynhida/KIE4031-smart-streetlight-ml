# Smart Street Lighting using Machine Learning

## Overview
This project applies machine learning techniques (Regression, Classification, Clustering) to optimize energy usage in smart street lighting systems.

## Models Used
- Regression (Energy Prediction)
- Classification (Power State)
- Clustering (Pattern Detection)

## Results
- R² Score: 0.62
- Accuracy: 83%
- Silhouette Score: 0.12

## Tools
- Python
- Scikit-learn
- Pandas, Matplotlib
